# Hello World HTML

Este é um exemplo simples de um documento HTML que exibe a mensagem **"Hello World"** em uma página da web. Ele é ideal para iniciantes aprenderem sobre a estrutura básica de um arquivo HTML.

## Estrutura do Código

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Hello</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>
